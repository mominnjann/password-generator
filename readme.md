# Strong Password Generator

A small Vite + React + TypeScript project implementing a Strong Password Generator UI with Tailwind CSS and shadcn-style components.

Technologies
- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-style UI primitives (local)

Features
- Customizable length (8–32)
- Toggle uppercase, lowercase, numbers, symbols
- Password strength indicator (Weak / Medium / Strong)
- One-click copy with visual feedback
- Auto-generate password on load
- Dark theme with purple/pink gradient accents, glassmorphism, and smooth transitions

Run locally
1. Install dependencies: npm install
2. Start dev server: npm run dev
3. Open the URL shown in the console (e.g., http://localhost:5174)

Notes
- The project uses local shadcn-style UI primitives for a clean modern look—feel free to replace with official shadcn-ui components if desired.
